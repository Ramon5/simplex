## Algoritmo Simplex ##

### Requisitos

Linux
Python >= 3

### Instruções ###

baixe o projeto:

$ git clone 

crie o ambiente virtual e ative:

$ virtualenv -p python3 .env
$ source .env/bin/activate

Instalando os requisitos:

$ pip install -r requirements.txt

### Executando ###

$ python3 view.py
